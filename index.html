Firebase

Sincronização

Para sincronizar entre computador e iPhone, preencha `firebase-config.js` com as chaves do seu projeto Firebase e publique as regras do `firestore.rules`.

Enquanto isso, o app usa dados de demonstração neste navegador.

Modelo

Como os dados ficam

users/{uid}/items/{itemId} Cada usuário enxerga somente seus próprios lançamentos.
items.paidOccurrences Marcar como pago remove a ocorrência da projeção pendente sem apagar histórico.
